##vidanta-world-with-surfly

This repository demonstrates an example integration between CXone Chat and Surfly, enabling interoperability between the two platforms.

It also includes upcoming changes required for a planned bot integration. These changes disable the chat text input when a conversation is assigned to the NLU bot, preventing unnecessary fallback responses caused by free-text input.

Rather than requiring two separate script updates, both changes are demonstrated together within this demo page for easier reference and testing.
